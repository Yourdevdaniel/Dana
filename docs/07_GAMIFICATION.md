# Gamification

## Objetivo

Dar progresso real baseado em dados da conta.

## Estrutura

- conquistas desbloqueadas
- conquistas em progresso
- conquistas bloqueadas
- badges conquistados
- ranking
- historico de XP

## Regras

- conquistas precisam refletir dados reais
- badge e conquista nao podem depender de valor falso
- tela precisa mostrar progresso e bloqueio
- o perfil pode destacar badges selecionadas

## Categorias

- base
- perfil
- fluxo
- metas
- dividas
- contas fixas
- casal
- patrimonio
- XP
- rede
- dashboard
