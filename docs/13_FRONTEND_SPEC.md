# Frontend Spec

## Entrada

- `frontend/src/main.tsx`
- `frontend/src/App.tsx`

## Responsabilidades

- login e cadastro
- verificacao de email
- login com Google
- dashboard solo e casal
- carteira, transacoes, categorias, metas, dividas, contas fixas
- conquistas
- perfil
- rede

## Estrutura

- `frontend/src/components/ui/`
- `frontend/src/data/achievements.ts`
- `frontend/src/lib/api.ts`
- `frontend/src/lib/utils.ts`

## Regras de UI

- mobile precisa ser funcional
- sidebar deve virar drawer no mobile
- telas devem caber sem sobreposicao
- erro tecnico nao deve aparecer para o usuario
