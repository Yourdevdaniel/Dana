# Project Charter

## Nome

Finance Couple

## Objetivo

Aplicacao financeira para casais com:

- autenticao com email e Google
- verificacao de email
- carteira e transacoes
- metas, dividas e contas fixas
- dashboard individual e do casal
- conquistas, badges e ranking
- perfil com foto e configuracao

## Principios

- dados reais do servidor
- sem dados falsos em telas principais
- fluxo unico entre web e futuro mobile
- seguranca e rastreabilidade
- documentacao sempre sincronizada com o codigo

## Responsabilidades

- `Codex`: frontend, experiencia, responsividade, consumo de API
- `Claude`: backend, regras de negocio, seguranca, persistencia, integracoes
