# Coding Standards

## Geral

- manter arquivos ASCII quando possivel
- nomes claros e curtos
- evitar abstração sem necessidade
- atualizar docs quando mudar comportamento

## Frontend

- seguir padrao de componentes existentes
- manter formularios controlados
- tratar erros de rede e autenticacao
- responsividade obrigatoria

## Backend

- manter separacao por app
- usar serializers, views, services e repositories conforme o projeto
- preservar o envelope de resposta
- nao expor dados sensiveis
