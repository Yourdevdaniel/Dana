# Code Reference

## Frontend

### Entry points

- `frontend/src/main.tsx`
- `frontend/src/App.tsx`
- `frontend/src/lib/api.ts`
- `frontend/src/data/achievements.ts`

- auth flow
- session persistence
- page shell
