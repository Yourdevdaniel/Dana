# Financial Engine

## Responsabilidade

Calcular visao financeira do usuario e do casal.

## Saidas principais

- saldo
- patrimonio liquido
- gasto medio mensal
- reserva recomendada
- tendencia mensal
- risco financeiro

## Entradas

- wallet
- salaries
- transactions
- goals
- debts
- fixed expenses
- couple group

## Regras

- dashboard individual usa os dados do usuario
- dashboard do casal consolida os membros
- metas, dividas e despesas fixas influenciam os indicadores
