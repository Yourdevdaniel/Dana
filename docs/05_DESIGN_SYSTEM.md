# Design System

## Visual style

- utilitario
- denso
- legivel
- sem enfeite desnecessario

## Principios

- cards com raio curto
- layout responsivo
- botao com icone quando fizer sentido
- texto nunca pode estourar o container
- usar estados vazios claros
- usar feedback de carregamento e erro

## Responsividade

- mobile precisa ter acesso a todas as areas principais
- sidebar vira drawer no mobile
- listas viram colunas verticais em telas estreitas
- formularios devem ser empilhados
